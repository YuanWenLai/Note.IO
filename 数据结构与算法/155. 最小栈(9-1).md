## [155. 最小栈](https://leetcode-cn.com/problems/min-stack/)

### 难度简单

设计一个支持 push，pop，top 操作，并能在常数时间内检索到最小元素的栈。

+ push(x) -- 将元素 x 推入栈中。

+ pop() -- 删除栈顶的元素。

+ top() -- 获取栈顶元素。

+ getMin() -- 检索栈中的最小元素。

  示例:

  ```JavaScript
  MinStack minStack = new MinStack();
  minStack.push(-2);
  minStack.push(0);
  minStack.push(-3);
  minStack.getMin();   --> 返回 -3.
  minStack.pop();
  minStack.top();      --> 返回 0.
  minStack.getMin();   --> 返回 -2.
  ```

  

## 解题思路

+ 维护两个栈data、minArr，一个存储数据，一个存储最小值

+ 每次元素value进栈A，判断，
  + 若minArr为空，第一个元素进栈minArr，
  + 若minArr不为空，判断minArr 的栈顶元素与value的大小
    + 若value小于minArr的栈顶元素，value进栈minArr
    + 否则不执行此次判断操作

+ 每当data栈有元素出栈时，判断这个出栈元素是否与minArr的栈顶元素相等
  + 若相等，此元素也要从minArr栈中出栈
  + 否则，不操作minArr栈
+ 调用getMin（），获得minArr的栈顶元素，就是最小栈



```javascript
/**
 * initialize your data structure here.
 */
var MinStack = function() {
    this.data = []
    this.minArr = []
};

/** 
 * @param {number} x
 * @return {void}
 */
MinStack.prototype.push = function(x) {
  this.data.push(x)
  let len = this.minArr.length
  if(len==0 || x <= this.minArr[len-1]){
    this.minArr.push(x)
  }
};

/**
 * @return {void}
 */
MinStack.prototype.pop = function() {
  let popVal = this.data.pop()
  let len = this.minArr.length
  if(popVal == this.minArr[len-1]){
    this.minArr.pop()
  }
  return popVal
};

/**
 * @return {number}
 */
MinStack.prototype.top = function() {
    let len = this.data.length
    return this.data[len-1]
};

/**
 * @return {number}
 */
MinStack.prototype.getMin = function() {
  let len = this.minArr.length
  if(len == 0){
    return null
  }
  return this.minArr[len-1] 
};

 */
```
