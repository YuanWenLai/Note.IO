## webpack自动清理构建目录产物

webpack每次构建时，不会清理上次遗留下来的打包文件，造成构建输出目录的output文件越来越多。

### 1、通过npm scripts命令来清理构建目录

- `rm -rf ./dist && webpack`



### 2、自动清理目录结构

```js
module.exports = {
    entry: {
        app: './src/app.js',
        search: './src/search.js'
    },
    output: {
        filename: '[name][chunkhash:8].js',
        path: __dirname + '/dist'
    },
    plugins:[
        new CleanWebpackPlugin() // 每次构建前，会自动清理dist文件夹下的文件
    ]
};
```
