## 常规语言和文件解析

### 1、资源解析: `ES6`

```js
// webpack.config.js
const path = require('path');
mode: 'development',
    entry: {
        main: path.resolve(__dirname,'../src/main.js'),
    },
        // 输出口
    output: {
        path: path.resolve(__dirname,'../dist')
    },
    module: {
        rules: [
            // 处理babel，es7
            {
                test: /\.js$/,
                use:[
                    {
                        loader: 'happypack/loader'
                    }
                ],
                exclude: /node_modules/
            },
            ]
        }
};

//babel的配置文件是 .babelrc
// 增加ES6的babel preset配置
{
    "preset": [
        "@babel/preset-env"
    ],
    "plugins":[
        "@babel/proposal-properities"
    ]
}
```



### 2、资源解析:`CSS`和`Sass`

```js
// webpack.config.js
const path = require('path');
mode: 'development',
    entry: {
        main: path.resolve(__dirname,'../src/main.js'),
    },
    // 输出口
    output: {
        path: path.resolve(__dirname,'../dist')
    },
    module: {
        rules: [
            {
                test: /\.css$/,
                use:[
                    'style-loader',
                    'css-loader'
                ]
            },
            {
                test: /\.scss$/,
                use:[
                    'style-loader',
                    'css-loader',
                    'sass-loader'
                ]
            },
            ]
        }
};
```



### 3、资源解析:图片文件 

- `url-loader` 也可以处理图⽚和字体,可以设置较⼩资源⾃动 `base64`
- `file-loader` 也可以⽤于处理字体

```js
// webpack.config.js
const path = require('path');
mode: 'development',
    entry: {
        main: path.resolve(__dirname,'../src/main.js'),
    },
    // 输出口
    output: {
        path: path.resolve(__dirname,'../dist')
    },
    module: {
        rules: [
            {
                test: /\.(png|svg|jpg|gif)$/,
                use:[
                    'url-loader'
                ]
            },
            {
                test: /\.(woff|woff2|eot|ttf|otf)$/,
                use:[
                    'file-loader'
                ]
            },
            ]
        }
};
```
